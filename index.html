How to fetch mutual fund NAVs

For stocks / ETFs: Use the NSE symbol — e.g. RELIANCE, HDFCBANK, NIFTYBEES. Fetched via Yahoo Finance (.NS suffix).

For mutual funds: Enter the fund's AMFI Scheme Code as the symbol (e.g. 120503 for Axis Bluechip). NAV is fetched directly from AMFI's official API — no proxy needed, always up to date.

Find your scheme code at amfiindia.com → NAV History → search your fund name.

Manual entry always works — click any price field to edit. For tracking only, not financial advice.
